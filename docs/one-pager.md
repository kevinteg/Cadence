# Cadence — One Pager (v2)

*The rhythm you return to.*

---

**What it is:** A cognitive operating system that lives inside your terminal. One
AI agent with three modes — Steward, Guide, Narrator — that manages your
attention, protects your flow state, and generates the narrative of what you're
building across work, family, and personal growth.

**What it replaces:** OmniFocus, scattered task lists, the mental overhead of
remembering where you left off. Not by being a better task app, but by being an
active partner that holds your context across sessions and pursuits.

**The core loop:**

```
  Enter session → Recap ("previously on...") → Flow state (silent, protected)
       → Breakpoint (nudges, quick wins, wellbeing) → Mark (save point)
       → Exit or switch → Next session picks up from marker
```

**The hierarchy:**

| Level | What it is | Example |
|-------|-----------|---------|
| **Pursuit** | An intentional commitment tied to your values | "Establish AI Test Framework at Nexthop" |
| **Project** | A scoped effort with a Definition of Done | "Stand Up CI Pipeline" |
| **Action** | The atomic thing you do | "Configure GitHub Actions workflow" |

**The weekly ritual — Reflect (~30 min):**

*Phase 1 — Get Clear:* Process triage decisions, clear 2-minute items, review
reconciler flags, confirm all active projects are relevant.

*Phase 2 — Get Focused:* Recap → Learn what worked → Triage WIP → Commit to
ONE leveraged priority that defines next week's win.

**Key behaviors:** Curated session entry, thought capture with transparent
triage, the 2-minute rule, WIP management, markers as save points, waiting-for
with reconciler, nudges batched at breakpoints, narratives from real data,
Definition of Done as a first-class managed checklist.

**Technical stack:** Local-first (SQLite + markdown), CLI-native (Claude Code),
TypeScript for integrations, Python for AI pipelines, agent-agnostic design.

**Who it's for:** Technical leaders juggling high-stakes work, a growing team,
family, personal projects, and a career transition — people who are productive
but disoriented, who finish weeks unable to articulate what moved forward.

---

*Cadence — one voice, one rhythm, one system that holds the whole story.*
