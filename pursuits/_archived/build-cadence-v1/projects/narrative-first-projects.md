# Narrative First Projects

Refactor the project model: replace the Definition of Done checklist with a narrative '## Intent' section that absorbs both motivation and felt-sense done-ness. Across 30+ projects in this repo, DoD has functioned as a redundant action list — bookkeeping overhead without insight. The new model: project body is Intent + Actions only; done-ness is judged through dialogue ('does the intent feel achieved?') rather than a checkbox sweep. Scope of THIS project is design-doc updates only; CLI/skill/reconciler changes are deferred to a follow-up project (scope captured in Notes).
