External-source ingestion for users coming from existing systems and for processing reading-list backlogs. Two related modes: (1) Tool migration — cadence import --from notion|things|omnifocus|... pulls existing tasks/projects into Cadence as pursuits/projects/captures, removing the empty-repo barrier for new users. (2) Reading-list ingestion — book recommendations, articles, podcasts get imported as captures or seeds, so the reading list is no longer separate from the cognitive operating system. Both deferred from v1.1: tool migration is downstream of having external users at all, reading-list ingestion is a quality-of-life enhancement for sustained use. Both fit naturally under the same surface (cadence import) with different --from values.
