# Improve Command Discoverability

## Intent

After a session start, Cadence prints a dashboard and a single "Next:" line — not enough scaffolding for someone who hasn't memorized the 15-verb surface. Make Cadence navigable from the dashboard alone via three additions: contextual next-step hints in the dashboard (2-3 ranked suggestions based on state); inline action menus on every `/status` drill-down listing the verbs that apply to the viewed entity; and a new `/cadence:help` skill that renders verb groupings (Diverge / Execute / Reflect / Setup / Browse) so users can browse the surface without reading skill files. Search and cross-verb numbered selection are deferred to a follow-up — discoverability is the priority. Done when a fresh-session user can move from dashboard → pursuit → project → start work, prompted by inline hints the whole way, without consulting external docs.
